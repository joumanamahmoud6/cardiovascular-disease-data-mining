# Cardiovascular Disease — Data Mining Project

## Overview
A data mining project on a cardiovascular disease dataset (1200 patients, 12 features)
applying clustering, fuzzy inference, and evolutionary optimization.

## Techniques Used
- **Preprocessing** — outlier removal, BMI engineering, StandardScaler
- **EDA** — 6 visualizations revealing key risk factors
- **Genetic Algorithm** — feature subset selection using KNN fitness
- **K-Medoids Clustering** — patient risk group discovery (PAM algorithm)
- **Hierarchical Clustering** — dendrogram analysis (Ward, Complete, Average)
- **Fuzzy Inference System** — rule-based cardiovascular risk scoring
- **System Pipeline** — integrated risk assessment for new patients

## Dataset
Cardiovascular Disease Dataset — Kaggle (1200 instances, 12 features)

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Place `mining_dataset.csv` in the same folder as the notebook

3. Run all cells top to bottom in `Data_Mining_Project.ipynb`
